A plugin to integrate WhiteLabelShopping into any Wordpress Blog in a simple and easy to use way. 

== Description ==

At WhiteLabelShopping we provide our customers with a high quality, highly indexable and responsive price comparison website. Our clients are usually blog and content site owners looking for an additional revenue stream. 

We integrate with the main players in the Price Comparison industry including: 

* Kelkoo 
* Pricegrabber
* Become
* Pricerunner
* Affiliate Window
* Connexity
* Twenga

This plugin will connect to your whitelabelshopping.net account and allow you to include our shopping technology into your blog for a more flexible experience. 

== Installation ==

1. Create and account here: whitelabelshopping.net (you will not be able to use this plugin without an account and domain setup)
1. Setup a new domain and input your shopping details. At this point you can either create a subdomain or use our export functionality. 
1. After you have an account install the plugin the usual way
1. After you've activated the plugin make sure you add you Export URL into the WLS Settings
1. That's it, you will now be able to either add products to the end of blog posts, add a widget for shopping and/or have a seperate shopping page.

== Frequently Asked Questions ==

1. Must i have a whitelabelshopping.net account to use this plugin?

In short, yes you will need to have an account. This plugin interacts with Whitelabelshopping.net to query and render relevant products to your blog posts. 

2. How do you pick what to search for on blog posts?

We take the tags from the post and perform a search. We will always randomise the search based on the tags on the post. 

3. Do you have any widgets? 

Yes we do, there is a MPU widget that will render results based on predefined searches and or allow the current user to perform an actual search. 

4. Do i need to have an account within Shopping.com / Twenga / Pricegrabber

Yes, you will need to have your own account with one of our price comparison providers. The integration to whitelabelshopping.net is easy and setup.
